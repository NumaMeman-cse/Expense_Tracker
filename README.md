# Expense Tracker

A sleek **dark-themed expense tracker** that allows users to track their daily, weekly, and monthly expenses with interactive charts.

---

## Features

- Add, edit, and delete expenses  
- Track total spending  
- View spending by category with pie charts  
- Weekly spending visualization with line charts  
- Dark aesthetic design inspired by modern UI trends  

---

## Technologies Used

- **Backend:** Python, Flask, SQLite  
- **Frontend:** HTML, CSS, JavaScript, Chart.js  
- **Other:** Git for version control
